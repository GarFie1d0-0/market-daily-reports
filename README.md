# market-daily — Proma Agent Skill

全球金融市场深度日报生成器。覆盖 A 股、港股、美股、日股、加密货币五大市场，采用 **5 角色 AI 投资委员会架构** 提供多维独立分析与制衡验证。

## v2.0 核心升级

**5 角色 AI 投资委员会** — 4 位独立分析师 + 1 位风控官并行工作，用角色分工制造"认知多样性"，用风控官角色防止群体思维：

| 角色 | 职责 |
|------|------|
| 宏观策略师 | 全球流动性、央行政策、地缘、跨市场共振 |
| A 股深度分析师 | 技术面 + 情绪 + 板块 + 龙虎榜 + 机会与风险 |
| 海外市场分析师 | 港股 + 美股 + 日股独立分析 |
| 加密分析师 | BTC/ETH 技术面 + 链上数据 + 鲸鱼 + 监管 |
| **风控与交叉验证官** | 审查其他 4 人，找矛盾、堵漏洞、推演极端尾部情景 |

**7-Phase 流程**：

```
Phase 1: 并行数据采集 (15-20 搜索)   → 素材收集
Phase 2: 多 Agent 并行深度分析  ★核心 → 5 角色 AI 投资委员会
Phase 3: 主 Agent CIO 整合    ★新增  → 通读 5 份报告 + 矛盾地图
Phase 4: 质量检查清单                  → 数据完整性与指标覆盖审查
Phase 5: 多源交叉验证与分歧展示        → 优先引用风控官发现
Phase 6: 报告编译                     → 含 AI 投资委员会观点全景
Phase 7: 双路径存档与版本控制          → 主路径 + Git 备份
```

**风控官硬性约束**：
- 必须至少找出 3 条矛盾
- 必须检测回声室效应
- 必须推演 1-2 个极端尾部情景
- 可以不同意任何分析师，但必须给出理由

## 报告输出

- **主路径**: `E:\Projects\dailyreports\YYYY-MM-DD-HHmmss.md`
- **Git 备份**: `workspace-files/reports/YYYY-MM-DD.md`
- 报告含：一分钟速览、技术分析、多空分歧展示、AI 投资委员会观点全景、事件日历、风险矩阵、极端尾部情景

## 结构

- `skill/SKILL.md` — Skill 定义文件（v2.0.0）
- `reports/` — 每日报告存档（不纳入版本控制）
- `.gitignore` — 排除敏感文件和报告

## 触发方式

在 Proma Agent 中手动调用，说"市场日报" / "market daily" / "全球市场报告" 即可。
